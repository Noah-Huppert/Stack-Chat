#Flavors
Flavors are based on Android versions. The name of the flavor is determined by the first part of the name for the lowest and highest Android Api version supported.

| Android Version | Flavor   |  
| --------------- | -------- |
| 21              | pop      |
| 16 - 19         | jellykit |
